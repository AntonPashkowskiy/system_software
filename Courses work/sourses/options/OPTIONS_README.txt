Исходые коды типов которые предназначены для обеспечения терминального интерфейса Linux.
Эти типы обеспечивают проверку аргументов, парсинг аргументов в опции архиватора, и последующую проверку
опций архиватора ( существование директорий и файлов в указанных путях и т.д ).
argument_handler(.cpp/.h) - обработка аргументов коммандной строки
archive_options.h - описывает тип опции для архиватора
switcher_task(.cpp/.h) - конечная обработка опций архиватора и выполнение задач опичанных в опциях.
